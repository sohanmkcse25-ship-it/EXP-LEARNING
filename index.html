HTML Tutorial: Code playground

Explanation

HTML is built using elements (tags) and attributes (extra details inside tags). Below are some basic tags you must know:

<!DOCTYPE html> → Declares the document type (compulsory at the top).

<html>...</html> → Root element that wraps the entire page.

<head>...</head> → Contains metadata, title, and linked styles/scripts.

<title>My Page</title> → Sets the page title shown in the browser tab.

<body>...</body> → Holds all visible content of the webpage.

<h1>Heading</h1> → Main heading element.

<p>Paragraph text</p> → Paragraph element for text.

<a href="https://example.com">Visit Site</a> → Link element with href attribute.

<img src="image.jpg" alt="Description"> → Image element with src and alt attributes.

<ul><li>Item</li></ul> → Unordered list with bullet points.

<table border="1">...</table> → Table element with border attribute.

<input type="text" placeholder="Enter name"> → Input element with type and placeholder.

<div style="color:blue;">Content</div> → Div element with style attribute for inline CSS.
